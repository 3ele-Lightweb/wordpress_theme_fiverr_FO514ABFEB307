Registers custom post types and custom fields for the Life_In_Balance theme

== Description ==

**The Life_In_Balance Toolbox plugin is meant to be used only with the Life_In_Balance WordPress theme.**

This plugin registers custom post types and custom fields that are needed in the [Life_In_Balance theme](http://wordpress.org/themes/sydney/)

== Installation ==

1. Download the folder 
2. Connect to FTP and access the files of the theme
3. Goto {root_folder_of_wordpress_installation}/wp-content/plugins
4. Place the downloaded folder inside that folder
5. Refresh AdminPanel and check installed Plugins
